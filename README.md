# dakoku

Money Forward attendance CLI

## Install
```shell
$ git clone https://github.com/ksk001100/dakoku
$ cd dakoku
$ cargo install --path .
$ cp .env.example ~/.dakoku
$ vim ~/.dakoku # Edit environment variables
```

## Usage
```shell
$ dakoku --help
$ dakoku attendance
$ dakoku a
$ dakoku leaving
$ dakoku l
```
